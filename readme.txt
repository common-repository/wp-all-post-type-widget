WP All Post Type Widget WordPress plugin add default post type and his category (post)

== Description ==
WP All Post Type Widget WordPress plugin add default post type and his category (post).
You can display post with custom post type and his category base.

= Descriptions of Widgets =

= Post Type (Custom Post Type or default) =

display a list of the custom posts by post type.

* **Title** - description that appears over the list of custom posts.
* **Post Type** - if selected, filter by a custom post type. (e.g. post).
* **Category** - Category filtered by post type.
* **Number of posts to show (at most unlimited)** - enter the number of posts to display.
* **Display post date?** - if checked, display post date.
* **Orderby** Default order by date
* **Order** Default order is desc


== Installation ==
1. Download plugin file wp-all-post-type-widget.zip, extract and upload folder wp-all-post-type-widget to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress.
3. Adds widgets to a widget area and configure settings through the 'Widgets' menu in WordPress.
4. Have fun!
